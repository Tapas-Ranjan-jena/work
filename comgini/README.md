# Comgini

## React + TypeScript + Vite Dashboard

This project is built using React, TypeScript and Vite.

It provides a dashboard structure with reusable layouts, masters modules,
bulk sender pages and requested documents features.

---

### Tech Stack
- React
- TypeScript
- Vite
- Bootstrap

---

### Development

Install dependencies:

npm install

Run project:

npm run dev

Build project:

npm run build
